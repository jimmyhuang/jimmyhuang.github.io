Projects

Prepare to sing and dance your way to stardom
Dance Dance Pose

Dance Dance Pose puts your singing and dancing skills to the test by utilizing motion capture and pitch detection. The bar is set high - you're competing against the artists of your favorite songs.

  • Real-time audio web detection and pose detection through Posenet, a machine learning model, ported over to Tensorflow JS
  • Tracks collisions on multiple planes: physical and spacial, as well as audio through pitch
  • Complete full-stack application on an Agile team using Travis CI for continuous integration
  • Github Demo

Take control of your car with an iOS mobile app
Tesla Remote

A personal project to contribute to the Tesla community. This mobile app allows you to view your car's telemetry data as well as perform actions like honking a horn, or flashing headlights

  • Created using React Native and renders clean interfaces on a mobile screen
  • Accesses the unofficial Tesla API through OAuth 2.0 Password Grant to provide security for users
